# Exploratory-Data-Analysis-EDA
# Dataset Description

## Dataset Source
The dataset used in this project is the Early Stage Diabetes Risk Prediction Dataset from the UCI Machine Learning Repository.

## Number of Rows and Columns
The dataset contains 520 rows and 17 columns.

## Features Description
The dataset includes several demographic and symptom-based features such as:
- Age: Age of the individual
- Gender: Male or Female
- Polyuria: Frequent urination (Yes/No)
- Polydipsia: Excessive thirst (Yes/No)
- Sudden weight loss: (Yes/No)
- Weakness: (Yes/No)
- Polyphagia: Excessive hunger (Yes/No)
- Genital thrush: (Yes/No)
- Visual blurring: (Yes/No)
- Itching: (Yes/No)
- Irritability: (Yes/No)
- Delayed healing: (Yes/No)
- Partial paresis: (Yes/No)
- Muscle stiffness: (Yes/No)
- Alopecia: Hair loss (Yes/No)
- Obesity: (Yes/No)

Target variable:
- Class: Indicates whether the individual is diabetic (Positive) or non-diabetic (Negative)

## Purpose of Using the Dataset
The purpose of this dataset is to analyze the factors associated with diabetes and build a machine learning model for early prediction of diabetes.
